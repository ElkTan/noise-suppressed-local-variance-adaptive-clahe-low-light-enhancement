# **Step 1：明确研究问题**

## **1.1 研究对象**

研究对象是：

低照度图像增强中的 CLAHE 噪声放大问题。

不要泛化成“所有低照度增强”或“复杂图像复原”。论文重点只讨论：

- 固定参数 CLAHE 为什么会放大暗区噪声；
- 如何根据局部结构强度自适应调节 clip limit；
- 如何避免噪声误判、块效应和亮度增强后的发灰问题。

------

## **1.2 写清楚传统 CLAHE 的不足**

传统 CLAHE 的核心参数是 clip limit。通常整幅图像使用固定参数：

C_{i,j}=C

问题是：

- 平坦暗区：不需要强增强，否则噪声被放大；
- 纹理区域：需要较强增强，否则细节仍然不明显；
- 固定 C 无法同时适配这两类区域。

因此提出：

C_{i,j}=f(s_{i,j})

其中 s_{i,j} 表示第 (i,j) 个图像块的结构强度。

## **5.1 RGB 转 YCbCr**

```python
ycrcb = cv2.cvtColor(img_rgb, cv2.COLOR_RGB2YCrCb)
Y, Cr, Cb = cv2.split(ycrcb)
```

注意 OpenCV 的顺序是：

```text

```

不是 Y, Cb, Cr。

------

## **5.2 平滑亮度图**

```python
Y_smooth = cv2.GaussianBlur(Y, (5, 5), 0)
```

用于计算结构方差。

------

## **5.3 分块计算方差**

假设 tile grid 是 8\times 8。

对每个 tile 计算：

v_{i,j}=\operatorname{Var}(\tilde{Y}_{i,j})

------

## **5.4 估计噪声方差**

使用 Laplacian + MAD：

H = \Delta Y

\sigma_n = \frac{\operatorname{median}(|H-\operatorname{median}(H)|)}{0.6745}

```python
lap = cv2.Laplacian(Y.astype(np.float32), cv2.CV_32F)
sigma_n = np.median(np.abs(lap - np.median(lap))) / 0.6745
sigma_n2 = sigma_n ** 2
```

------

## **5.5 构造结构强度**

s_{i,j}=\max(v_{i,j}-\lambda\sigma_n^2,0)

建议：

\lambda=0.5

------

## **5.6 计算自适应参数** k

k=\beta\operatorname{mean}(s_{i,j})+\epsilon

建议：

\beta=1.0,\quad \epsilon=10^{-6}

------

## **5.7 计算 raw clip limit**

C_{i,j}^{raw} = C_{\min} + (C_{\max}-C_{\min}) \frac{s_{i,j}}{s_{i,j}+k}

建议：

C_{\min}=1.0,\quad C_{\max}=4.0

------

## **5.8 平滑 clip map**

C_{i,j}=G_\rho*C_{i,j}^{raw}

代码：

```python
clip_map = cv2.GaussianBlur(clip_raw.astype(np.float32), (3, 3), 0)
```

------

## **5.9 对每个 tile 执行 CLAHE**

注意：OpenCV 自带 `cv2.createCLAHE` 不支持每个 tile 用不同 clip limit。

你有两个实现选择：

### **简单实现方案**

对每个 tile 单独做 CLAHE，然后拼接。

缺点：边界插值不如 OpenCV 原版完美。

优点：实现简单，适合作业。

### **稍好实现方案**

每个 tile 做局部 histogram mapping，然后对相邻 tile 的映射结果进行双线性插值。

优点：更接近标准 CLAHE。

缺点：代码复杂。

**建议你采用简单实现方案 + clip map 平滑 + overlap blending。**

具体做法：

1. 每个 tile 用对应 C_{i,j} 做 CLAHE；
2. 对 tile 输出进行拼接；
3. 最后轻微双边滤波或高斯融合边界。

课程作业足够。

# **Step 6：色度补偿**

增强后得到 Y'。

计算亮度增益：

r(x)=\frac{Y'(x)+\epsilon}{Y(x)+\epsilon}

压缩增益：

g(x)=r(x)^\delta

建议：

\delta=0.5

然后补偿色度：

C_r'(x)=128+\eta g(x)(C_r(x)-128)

C_b'(x)=128+\eta g(x)(C_b(x)-128)

建议：

\eta=1.0

最后 clip 到 [0,255]。

## **7.3 暗区噪声放大比 NAR**

这是你的自定义指标，强烈建议保留。

定义低亮度低梯度区域：

\Omega=\{x\mid Y(x)<T_l,\ |\nabla Y(x)|<T_g\}

建议：

T_l=50

T_g=10

然后计算：

NAR= \frac{\operatorname{Var}(Y'_{\Omega})} {\operatorname{Var}(Y_{\Omega})+\epsilon}

NAR 越低，说明暗区噪声放大越少。

但注意：NAR 不能单独说明图像更好，因为完全不增强也会有低 NAR。所以它必须和亮度、SSIM 一起看。
